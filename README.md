# Vita Moda - Sito Web Landing Page

Sito web vetrina per la boutique "Vita Moda" ad Alberobello. Design minimalista stile Zara con accenti fucsia.

## Struttura del Progetto
- `index.html`: Home page principale.
- `styles.css`: Foglio di stile CSS.
- `script.js`: Logica JavaScript per interazioni e animazioni.
- `assets/`: Cartella contenente immagini, video e loghi.

## Istruzioni per lo Sviluppo
Aprire `index.html` nel browser o usare un server locale (es. Live Server di VS Code).

## Guida alla Consegna
Per istruzioni dettagliate su come mettere il sito online, consultare il file `DELIVERY_GUIDE.md` nella cartella di documentazione.
